# Local Agent Workflow

Before any analysis that may lead to code edits, and always before any file edit in this repository:

1. Read [`docs/ai-dev-prompt.md`](./docs/ai-dev-prompt.md) in full.
2. Treat it as the local project workflow and architecture policy.
3. In the first working update for the task, explicitly state that `docs/ai-dev-prompt.md` has been reviewed.
4. Do not start editing files until steps 1-3 are satisfied.

If there is any conflict, system, developer, and direct user instructions take precedence over this file.
